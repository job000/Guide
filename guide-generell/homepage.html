Moderne HomePage Guide

Hva du vil lage

En moderne, responsiv forside med:

  • Hero-seksjon med animasjoner og gradient
  • Feature-grid med hover-effekter
  • Social proof (testimonials/logoer)
  • Call-to-action seksjoner
  • Smooth scroll og micro-interactions

Moderne Designtrender 2024/2025

Hold deg oppdatert med disse trendene som gjor nettsider profesjonelle og engasjerende:

Anatomien til en Moderne HomePage

En effektiv forside folger denne strukturen:

1. Hero Section

Forste inntrykk. Stor headline, subheadline, CTA-knapper. Ma fange oppmerksomhet pa under 3 sekunder.

2. Value Proposition / Features

Vis hva du tilbyr. 3-6 n√∏kkelfunksjoner eller fordeler i et visuelt grid.

3. Social Proof

Bygg tillit. Kundelogoer, testimonials, tall/statistikk som viser resultater.

4. Call-to-Action

Konverteringspunkt. Tydelig handling du vil at besokende skal ta.

5. Footer

Navigasjon, kontaktinfo, sosiale medier, copyright.

Sticky Header med Blur

Navigasjon som endres ved scroll

// src/components/layout/Header.tsx
'use client'
import { useState, useEffect } from 'react'
import Link from 'next/link'
import Image from 'next/image'

const navLinks = [
  { href: '/tjenester', label: 'Tjenester' },
  { href: '/portfolio', label: 'Portfolio' },
  { href: '/om-oss', label: 'Om oss' },
  { href: '/blogg', label: 'Blogg' },
]

export default function Header() {
  const [isScrolled, setIsScrolled] = useState(false)
  const [isMobileMenuOpen, setIsMobileMenuOpen] = useState(false)

  useEffect(() => {
    const handleScroll = () => {
      setIsScrolled(window.scrollY > 50)
    }
    window.addEventListener('scroll', handleScroll)
    return () => window.removeEventListener('scroll', handleScroll)
  }, [])

  return (
    <header className={`
      fixed top-0 left-0 right-0 z-50 transition-all duration-300
      ${isScrolled
        ? 'bg-white/80 backdrop-blur-lg shadow-lg py-4'
        : 'bg-transparent py-6'
      }
    `}>
      <nav className="max-w-7xl mx-auto px-4 flex items-center justify-between">
        {/* Logo */}
        <Link href="/" className="relative z-10">
          <Image
            src={isScrolled ? '/logo-dark.svg' : '/logo-light.svg'}
            alt="Logo"
            width={140}
            height={40}
            className="transition-opacity"
          />
        </Link>

        {/* Desktop Navigation */}
        <div className="hidden md:flex items-center gap-8">
          {navLinks.map((link) => (
            <Link
              key={link.href}
              href={link.href}
              className={`
                font-medium transition-colors
                ${isScrolled
                  ? 'text-gray-700 hover:text-purple-600'
                  : 'text-white/90 hover:text-white'
                }
              `}
            >
              {link.label}
            </Link>
          ))}
          <Link
            href="/kontakt"
            className={`
              px-6 py-2.5 rounded-lg font-semibold transition-all
              ${isScrolled
                ? 'bg-purple-600 text-white hover:bg-purple-700'
                : 'bg-white text-purple-600 hover:bg-gray-100'
              }
            `}
          >
            Kontakt oss
          </Link>
        </div>

        {/* Mobile menu button */}
        <button
          onClick={() => setIsMobileMenuOpen(!isMobileMenuOpen)}
          className="md:hidden relative z-10"
        >
          <svg
            className={`w-6 h-6 ${isScrolled ? 'text-gray-900' : 'text-white'}`}
            fill="none"
            stroke="currentColor"
            viewBox="0 0 24 24"
          >
            {isMobileMenuOpen ? (
              <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M6 18L18 6M6 6l12 12" />
            ) : (
              <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M4 6h16M4 12h16M4 18h16" />
            )}
          </svg>
        </button>

        {/* Mobile menu */}
        {isMobileMenuOpen && (
          <div className="absolute top-full left-0 right-0 bg-white shadow-xl md:hidden">
            <div className="p-4 space-y-4">
              {navLinks.map((link) => (
                <Link
                  key={link.href}
                  href={link.href}
                  className="block py-2 text-gray-700 hover:text-purple-600 font-medium"
                  onClick={() => setIsMobileMenuOpen(false)}
                >
                  {link.label}
                </Link>
              ))}
              <Link
                href="/kontakt"
                className="block w-full text-center py-3 bg-purple-600 text-white rounded-lg font-semibold"
                onClick={() => setIsMobileMenuOpen(false)}
              >
                Kontakt oss
              </Link>
            </div>
          </div>
        )}
      </nav>
    </header>
  )
}
