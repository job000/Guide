Leveringsplan: 5 dager

Dag 1: Oppsett

Kjør auto-setup, konfigurer Supabase, test at alt fungerer.

curl -fsSL https://obligarconsulting.no/setup.sh | bash -s kundenavn

Dag 2: Sider & Layout

Opprett alle sider, tilpass header/footer, legg til navigasjon.

Dag 3: Funksjonalitet

Kontaktskjema, database-integrasjon, eventuelle spesialfunksjoner.

Dag 4: Styling

Mobile-first design, farger, bilder, finjustering.

Dag 5: Deploy & Test

Deploy til Vercel, test på mobil, lever til kunde.

Før du begynner

Krav:
  • Node.js 18+ installert (node --version)
  • VS Code med Tailwind IntelliSense
  • Supabase-konto (gratis på supabase.com)

Sliter du?

  • React basics - Komponenter, props, state

  • Feilmeldinger - Vanlige feil og løsninger
