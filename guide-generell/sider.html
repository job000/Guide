Lær hvordan du oppretter nye sider i Next.js 15 App Router. Inkluderer komplette eksempler for Om oss, Tjenester og Kontakt.

Slik fungerer routing i Next.js

App Router regler:
  • Hver mappe i src/app/ blir en URL-rute
  • Filen page.tsx i mappen er selve siden
  • Mappenavn = URL-path (f.eks. om-oss/ → /om-oss)
Fil URL Beskrivelse
src/app/page.tsx / Forsiden
src/app/om-oss/page.tsx /om-oss Om oss-siden
src/app/tjenester/page.tsx /tjenester Tjenester-siden
src/app/kontakt/page.tsx /kontakt Kontakt-siden
src/app/tjenester/[id]/page.tsx /tjenester/123 Dynamisk side (f.eks. enkelt tjeneste)

Steg 1: Opprett mapper

Kjør i terminalen:

mkdir -p src/app/om-oss src/app/tjenester src/app/kontakt src/app/personvern

Tips & Best Practices

SEO: Hver side bør ha unik metadata (title, description).
Bilder: Bruk next/image for automatisk optimalisering: 
import Image from "next/image"

<Image
  src="/images/team.jpg"
  alt="Vårt team"
  width={600}
  height={400}
  className="rounded-lg"
/>
Loading states: Opprett loading.tsx i mappen for å vise loading-spinner: 
// src/app/tjenester/loading.tsx
export default function Loading() {
  return <div className="text-center py-20">Laster...</div>
}
