CSS & Tailwind Klasser - Komplett Guide

Lær alle de viktigste CSS-klassene fra Tailwind med visuelle eksempler og kode du kan kopiere direkte.

Hva er Tailwind CSS?

Tailwind er et utility-first CSS-rammeverk. I stedet for å skrive egne CSS-filer, bruker du ferdiglagde klasser direkte i HTML.

Tradisjonell CSS vs Tailwind

Tradisjonell CSS Tailwind CSS 
/* styles.css */
.button {
  background-color: blue;
  color: white;
  padding: 8px 16px;
  border-radius: 4px;
}
 
<button class="bg-blue-500 text-white px-4 py-2 rounded">
  Klikk meg
</button>

Flexbox (display: flex)

Flexbox brukes for å lage fleksible layouts hvor elementer kan plasseres og justeres enkelt.

Grunnleggende flex

KlasseCSSBeskrivelse
flexdisplay: flexGjør elementet til en flex-container
inline-flexdisplay: inline-flexFlex-container som inline-element

Flex-retning (flex-direction)

KlasseCSSBeskrivelse
flex-rowflex-direction: rowElementer på rekke (standard)
flex-row-reverseflex-direction: row-reverseRekke, omvendt rekkefølge
flex-colflex-direction: columnElementer i kolonne (vertikalt)
flex-col-reverseflex-direction: column-reverseKolonne, omvendt

Responsive Design

Tailwind er mobile-first. Klasser uten prefix gjelder mobil, deretter legger du til for større skjermer.

Breakpoints

PrefixMinimum breddeEnhet
(ingen)0pxMobil
sm:640pxStor mobil
md:768pxTablet
lg:1024pxLaptop
xl:1280pxDesktop
2xl:1536pxStor skjerm

Eksempel: Responsivt layout

<div class="
  flex
  flex-col       /* Vertikal på mobil */
  md:flex-row    /* Horisontal fra tablet */
  gap-4
">
  <div class="md:w-1/3">Sidebar</div>
  <div class="md:w-2/3">Hovedinnhold</div>
</div>

Skjul/Vis elementer

<!-- Kun synlig på mobil -->
<div class="block md:hidden">Mobil-meny</div>

<!-- Kun synlig på desktop -->
<div class="hidden md:block">Desktop-meny</div>
