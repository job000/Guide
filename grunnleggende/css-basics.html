07. Styling med Tailwind CSS

Mobile-first design og konsistent styling med Tailwind CSS.

Se også: Komplett CSS/Tailwind klasseguide med alle klasser forklart med eksempler.

Oppsett

/* src/app/globals.css */
@import "tailwindcss";

:root {
  --primary: #3b82f6;
  --primary-foreground: #ffffff;
}

body {
  font-family: system-ui, -apple-system, sans-serif;
}

Mobile-First

Tailwind bruker mobile-first breakpoints. Start med mobil, legg til for større skjermer.

PrefixMin-breddeEnhet
(ingen)0pxMobil
sm:640pxStor mobil
md:768pxTablet
lg:1024pxLaptop
xl:1280pxDesktop

Eksempel

<div class="
  grid
  grid-cols-1        /* 1 kolonne på mobil */
  md:grid-cols-2     /* 2 kolonner fra 768px */
  lg:grid-cols-3     /* 3 kolonner fra 1024px */
  gap-4
">
  <div>Kort 1</div>
  <div>Kort 2</div>
  <div>Kort 3</div>
</div>

cn() hjelpefunksjon

// src/lib/utils.ts
import { clsx, type ClassValue } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}

// Bruk:
<div className={cn(
  "px-4 py-2 rounded",
  isActive && "bg-blue-500 text-white",
  !isActive && "bg-gray-100"
)}>
  Innhold
</div>
